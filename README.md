# search-app

## Page setup
```
cd search-app
```
### Install wrapper 
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

## API setup 
```
pip install uvicorn
```

### Runs server 
```
uvicorn API:app --reload
```
